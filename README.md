# CAB302 Week 3 – Address Book (JavaFX)

This project implements a simple Address Book application using **JavaFX** as part of the **CAB302 Week 3 Workshop**.

## Features
- JavaFX GUI application
- Terms and Conditions screen
- Address Book main screen
- Contact list display
- Full CRUD functionality:
  - Create contact
  - Read contact
  - Update contact
  - Delete contact

## Architecture
The application uses a simple in-memory database design.

Classes implemented:

- **Contact** – data model representing a contact
- **IContactDAO** – interface for CRUD operations
- **MockContactDAO** – in-memory implementation of the DAO
- **MainController** – handles the address book UI logic
- **HelloController** – handles the terms screen
